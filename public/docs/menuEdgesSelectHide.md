# Menu Edges

## Select 

A native capability of cytoscape. 
Select edges by click and shift-click. 

### status 
In the Edges main menu are informations about selected and hidden nodes  
***Edges  Selected/Visibles ( selected/hidden)*** 

### selection of edges related to selected nodes : *from selected nodes* 

***All / Incoming / outgoing*** decide which edges to retain.  
***edges connecting two nodes*** select (and show if hidden) the edges that relate two selected nodes


## Filter

Select edges with dynamic added category : ***must click on  "by category >"*** to retrieve available categories before a choice. 

## Hide 

obvious. 

## Label aspect

Show or hide the edge label on their respective lines. 

## data model 

Specific actions dealing with a database. 
See ***[quick tour ](./quickTour.md)*** for more information


## delete selected 

 remove definitively the edges from the graph.
